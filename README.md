# Navegacion entre 3 paginas

lenguaje dart flutter, nivel principiante, navegación entre 3 paginas utilizando rutas nombradas, desde main llamar a pagina1, en la página en appbar mostrar el texto "Angel Salinas 6J" en negro y naranja de fondo, iconos negros, en body un contenedor redondeado, color azul claro de 200x200 con texto blanco y centrado, y un boton color amarillo texto negro para sellecionar pagina 2, en la pagina 2 un appbar con el texto "segunda pagina 6J" en rojo, fondo negro y iconos en blanco, en body una imagen desde la red y un boton para avanzar a la pagina 3, en la pagina 3 un appbar con texto negro "pagina 3 Angel", colo de fondo verde claro, todo en un solo archivo.
